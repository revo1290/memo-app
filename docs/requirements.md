# メモ帳Webアプリ 要件定義書

## 1. プロジェクト概要

### 1.1 プロジェクト名
Memo App（仮称）

### 1.2 目的
シンプルで使いやすいメモ帳Webアプリケーションを提供する

### 1.3 ターゲットユーザー
個人でメモを管理したいユーザー

---

## 2. 技術スタック

| 項目 | 技術 |
|------|------|
| フレームワーク | Next.js 14+ (App Router) |
| 言語 | TypeScript |
| スタイリング | Tailwind CSS |
| データベース | Vercel Postgres |
| ORM | Prisma（予定） |
| デプロイ | Vercel |

---

## 3. 機能要件

### 3.1 基本機能（CRUD） ✅

| 機能 | 説明 | 状態 |
|------|------|------|
| メモ作成 | 新しいメモを作成できる | ✅ |
| メモ一覧表示 | 保存されたメモを一覧で表示する | ✅ |
| メモ編集 | 既存のメモを編集できる | ✅ |
| メモ削除 | メモを削除できる | ✅ |

### 3.2 マークダウン対応 ✅

- [x] メモ本文をマークダウン記法で記述可能
- [x] プレビュー表示でレンダリング結果を確認可能
- [x] 対応記法：見出し、リスト、コードブロック、リンク、太字/斜体など
- [x] GFM（GitHub Flavored Markdown）対応
- [x] シンタックスハイライト対応

### 3.3 タグ・カテゴリ機能 ✅

- [x] メモに複数のタグを付与できる
- [x] タグでフィルタリングして表示できる
- [x] タグの作成が可能（カラーピッカー付き）
- [x] タグの削除が可能（ホバー時に削除ボタン表示）
- [x] プリセットカラーによる簡単な色選択

### 3.4 検索機能 ✅

- [x] メモのタイトル・本文をキーワード検索できる
- [x] デバウンス処理によるスムーズな検索
- [x] URLパラメータによる検索状態の保持

### 3.5 お気に入り/ピン留め機能 ✅

- [x] メモをお気に入り登録できる
- [x] お気に入りメモは一覧の上部に固定表示
- [x] カード上でホバー時にピン留めボタン表示

### 3.6 並び替え機能 ✅

- [x] 作成日（新しい順/古い順）
- [x] 更新日（新しい順/古い順）
- [x] タイトル（A-Z/Z-A）

---

## 4. 非機能要件

### 4.1 認証
- 認証機能なし（誰でもアクセス可能）
- 将来的な拡張の可能性は考慮

### 4.2 パフォーマンス
- 初期表示は2秒以内
- メモの保存・更新は即座に反映

### 4.3 レスポンシブデザイン
- PC/タブレット/スマートフォンに対応

### 4.4 アクセシビリティ
- キーボード操作対応
- 適切なセマンティックHTML

---

## 5. 画面構成

### 5.1 メイン画面（一覧）
- ヘッダー（アプリ名、検索バー）
- サイドバー（タグ一覧、フィルター）
- メモ一覧（カード形式）
- 新規作成ボタン

### 5.2 メモ詳細/編集画面
- タイトル入力欄
- 本文入力欄（マークダウンエディタ）
- プレビュー切り替え
- タグ選択/追加
- お気に入りボタン
- 保存/キャンセルボタン

---

## 6. データモデル（案）

### 6.1 Memo（メモ）
| カラム | 型 | 説明 |
|--------|------|------|
| id | UUID | 主キー |
| title | VARCHAR(255) | タイトル |
| content | TEXT | 本文（マークダウン） |
| isPinned | BOOLEAN | ピン留め状態 |
| createdAt | TIMESTAMP | 作成日時 |
| updatedAt | TIMESTAMP | 更新日時 |

### 6.2 Tag（タグ）
| カラム | 型 | 説明 |
|--------|------|------|
| id | UUID | 主キー |
| name | VARCHAR(50) | タグ名 |
| color | VARCHAR(7) | 色コード（任意） |
| createdAt | TIMESTAMP | 作成日時 |

### 6.3 MemoTag（中間テーブル）
| カラム | 型 | 説明 |
|--------|------|------|
| memoId | UUID | メモID |
| tagId | UUID | タグID |
